- create folder `exp-tracking-dvc` with src and data without dvc pipeline.
- Instal DVC LIVE
```sh
pip install dvclive
pip show dvclive
python .\src\water_model.py # print the accuracy
```
- create params.yaml and modify the water_model.py code accordingly 
- use live.log_metric, live.log_param
```sh
git init
dvc init
dvc add .\data\water_potability.csv ## water_potability.csv.dvc will be created
git status
dvc status
git add .
git commit -m "Intial commit"
python .\src\water_model.py   
```
- Now check dvc.yaml file with metrics.json & params.yaml is available
```sh
dvc exp show
```
- Open D:\PersonalRepo\MLOPS\exp-tracking-dvc\params.yaml change the n_estimators value
- For better visibility install Extension **DVC**
- we can apply workspace so that n_estimators it will be choosen
```sh
dvc exp diff forty-jump noble-mast
dvc exp remove forty-jump
dvc exp apply noble-mast #check params.yaml and check
```
