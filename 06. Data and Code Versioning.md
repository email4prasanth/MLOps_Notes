- create code version git and data version dvc
```sh
git init          # only if not already a git repo
dvc init
# copy the required py files to respective folder paras.yaml file
# push the code
```
- To create a data version in local
```sh
# type window r hit %temp% copy the path create myremote(default storage) and dvc1 name
dvc remote add -d myremote C:\Users\LUCKYD~1\AppData\Local\Temp\dvc1
# perform dvc stage add --force and dvc repro
dvc push # To maintain data version
git push origin master # To maintain code version 
```
- To swith between versions
```sh
git log --oneline 

# move to previous version
git checkout <last before commit> # note the change in code data_processing.py
dvc checkout # note the change in processed folder

# switch to current version
git checkout <last commit> # note the change in code data_processing.py
dvc checkout # note the change in processed folder
```
- To clone code and data
```sh
git clone v2
dvc fetch
dvc checkout

git log --oneline 
# move to previous version
git checkout <last before commit> # note the change in code data_processing.py
dvc fetch

```
